## 🎯 О проекте

**PvZ (Plants Verification Size)** — веб-приложение с компьютерным зрением для автоматического биометрического анализа растений. Позволяет агрономам, селекционерам и исследователям за секунды измерять площадь листьев, длину корней и морфологию стеблей с точностью до ±3%.

### 🔑 Ключевые возможности
- 🤖 **Классификация:** Автоматическое определение типа растения (пшеница / руккола).
- 🎯 **Сегментация:** Попиксельное выделение частей растения (U-Net, YOLO).
- 📏 **Метрики:** Расчёт размеров в реальных единицах (мм, мм²) с калибровкой по маркеру.
- 📊 **Отчётность:** Экспорт результатов в CSV + визуализация маски сегментации.
- 🌐 **Веб-интерфейс:** Работа через браузер без установки ПО на клиенте.

### 👥 Целевая аудитория
- Агрономы, селекционеры, научные сотрудники.

- Малые и средние агробизнесы.

### Актуальность и значимость
- **Проблема:** Отсутствие доступных инструментов для быстрого и точного измерения биометрических параметров растений. Ручные измерения трудоёмки, субъективны и не масштабируемы.
- **Решение:** Автоматизация процесса анализа с помощью ИИ, снижение затрат времени на 90%.
- **Востребованность:** Продукт ориентирован на внедрение в образовательный процесс и малый агробизнес РФ.

### Новизна
- **Гибридная архитектура:** Комбинация YOLO (скорость) и U-Net (точность) с адаптивным выбором модели.
- **Адаптивная калибровка:** Пересчёт пикселей в мм по пользовательскому маркеру без фиксации расстояния камеры.
- **Локальная обработка:** Данные не уходят на сторонние серверы (конфиденциальность).
- **Отсутствие аналогов:** В отличие от дорогих систем (WinRHIZO), PvZ — бесплатное open-source решение с веб-интерфейсом.

### Доступность для целевой аудитории

- **Кроссплатформенность:** Работает на Windows, Linux, macOS.

## 💻 Быстрый старт (Установка)

### 1. Подготовка окружения
-   Убедитесь, что установлен **Python 3.12**.
-   Желательно наличие CUDA для ускорения обработки (но работает и на CPU).

### 2. Установка
```bash
git clone https://github.com/YaLapushok/Plants_Verefication_Size-PvZ-
cd Plants_Verefication_Size-PvZ-

# Создание и активация виртуального окружения
python -m venv venv
venv\Scripts\activate  # Для Windows
```

### 3. Установка зависимостей
```bash
pip install -r requirements.txt
```

### 4. Запуск приложения
```bash
uvicorn app.main:app --port 8000 --reload
```
После запуска откройте в браузере: [http://127.0.0.1:8000](http://127.0.0.1:8000)
