# Plants Verification Size (PvZ) - Telegram Bot

Этот репозиторий содержит модели для классификации и сегментации растений (пшеница и рукола), а также Telegram-бота, который автоматически обрабатывает входящие изображения:
1. Классифицирует растение на фото (Пшеница или Рукола).
2. Применяет соответствующую модель сегментации (на основе YOLO).
3. Отправляет пользователю результат с наложенной маской сегментации.

## Установка и настройка

1. **Создайте виртуальное окружение (рекомендуется):**
   ```bash
   python -m venv venv
   # Активация для Windows:
   venv\Scripts\activate
   # Активация для Linux/Mac:
   source venv/bin/activate
   ```

2. **Установите зависимости:**
   ```bash
   pip install -r requirements.txt
   ```

3. **Настройте переменные окружения:**
   - Пересохраните файл `.env.example` как `.env` (или просто создайте файл `.env`).
   - Откройте бота **@BotFather** в Telegram, создайте нового бота и получите токен.
   - Вставьте токен в файл `.env`:
     ```env
     TELEGRAM_BOT_TOKEN="ваш_токен"
     ```

## Использование

### ЗАПУСК ВЕБ-САЙТА (Новое!)
Мы добавили современное веб-приложение на FastAPI, чтобы вы могли анализировать изображения прямо в браузере с приятным интерфейсом:

1. Запустите сервер:
   ```bash
   uvicorn app.main:app --reload
   ```
2. Откройте в браузере: [http://127.0.0.1:8000](http://127.0.0.1:8000)
3. Загрузите фотографию (Drag & Drop) и дождитесь результата анализа с диаграммой!

### ЗАПУСК TELEGRAM-БОТА
Если вам удобнее пользоваться ботом:
1. Запустите бота (обратите внимание, теперь он лежит в папке `telegram`):
   ```bash
   python -m telegram.bot
   ```
   *(Команда запускается из корневой папки проекта).*
2. Откройте вашего бота в Telegram, отправьте ему команду `/start` и пришлите фото.
