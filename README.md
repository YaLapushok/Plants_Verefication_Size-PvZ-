# Plants Verification Size (PvZ) - AI Агрономия 🌿

О проекте:
Plants Verification Size - это интеллектуальная система для агрономов и исследователей, которая автоматически определяет виды растений, сегментирует их на изображениях и рассчитывает ключевые биометрические параметры.
Наш проект за секунды обрабатывает изображение, выделяет каждое растение и выдаёт точные метрики: площадь листьев, длину стеблей, статистику по кадру. Он решает проблему ручного анализа фотографий посевов - это долго, субъективно и трудно масштабируется.  

Функции:
- Классификация культур. Автоматическое распознавание пшеницы, руколы и других растений.
- Instance Segmentation. Точное выделение границ каждого объекта на изображении с помощью YOLOv26.
- Биометрический анализ: 
  - Расчёт площади листьев (в пикселях и условных см²)
  - Измерение длины и площади стеблей и корней
  - Статистика по количеству объектов на кадре
- Визуализация. Наложение масок, отрисовка bounding box-ов.
- Два интерфейса: 
  - Веб-интерфейс на FastAPI (удобно для десктопа)
  - Telegram-бот на Aiogram (доступ с телефона в поле)

## 🛠 Технический стек

| Компонент  | Технология                                  |
|------------|---------------------------------------------|
| Backend    | Python 3.9+, FastAPI, Uvicorn               |
| ML / CV    | PyTorch, Ultralytics YOLOv26, OpenCV, NumPy |
| Bot        | Aiogram 3.x                                 |
| Frontend   | Jinja2, HTML/CSS/JS                         |
| Data       | Pandas                                      |
| DevOps     | GitHub                                      |

Проект включает в себя современные технологии машинного обучения и веб-разработки:

- **Машинное обучение и Компьютерное зрение:**
  - `Ultralytics (YOLO)` — для классификации и инстанс-сегментации.
  - `OpenCV` и `scikit-image` — для обработки изображений и скелетизации.
  - `NumPy` и `SciPy` — для вычислений площадей и длины растений.
- **Backend & Веб-интерфейс:**
  - `FastAPI` — современный и быстрый асинхронный Python веб-фреймворк.
  - `Uvicorn` — ASGI-сервер.
  - `Jinja2` — шаблонизатор для HTML: одностраничное веб-приложение с премиальным UI (Glassmorphism, Vanilla CSS, JS).
  - `Matplotlib` — для генерации графиков статистики распределения биомассы.
- **Telegram бот:**
  - `Aiogram (3.x)` — асинхронный фреймворк для взаимодействия с API Telegram.

## 💻 Как запустить локально

### 1. Подготовка окружения
Склонируйте репозиторий и создайте виртуальное окружение:
```bash
git clone <ваш_URL_репозитория>
cd Plants_Verefication_Size-PvZ-

# Создание виртуального окружения
python -m venv venv

# Активация для Windows:
venv\Scripts\activate
# Активация для Linux/Mac:
source venv/bin/activate
```

### 2. Установка зависимостей
```bash
pip install -r requirements.txt
```

### 3. Настройка переменных окружения
- Создайте файл `.env` в корневой папке (можно скопировать из `.env.example`).
- Получите токен у **@BotFather** в Telegram и вставьте его в `.env`:
```env
TELEGRAM_BOT_TOKEN="ваш_токен"
```

### 4. Запуск Веб-интерфейса
Для запуска красивого локального сайта выполните:
```bash
uvicorn app.main:app --reload
```
После запуска перейдите в браузере по адресу: [http://127.0.0.1:8000](http://127.0.0.1:8000).

### 5. Запуск Telegram-бота
Чтобы бот начал отвечать в Telegram, запустите его скрипт из корня проекта:
```bash
python -m telegram.bot
```
